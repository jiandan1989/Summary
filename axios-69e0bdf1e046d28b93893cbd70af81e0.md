### axios

- vue 设置代理
  - 配置 host

  localhost                   www.exaple.com

```js
proxy: {
   '/apis': {    //将www.exaple.com印射为/apis
      target: 'https://www.exaple.com',  // 接口域名
      changeOrigin: true,  //是否跨域
      pathRewrite: {
        '^/apis': ''   //需要rewrite的,
      }
    }
  }
```
